Isolation Architecture

Why Database-Per-Tenant is the Only Acceptable Standard

In shared-table SaaS architectures, a single missing WHERE tenant_id = X clause can expose your entire company's payroll, tasks, and confidential HR data to a competitor. We eliminated this risk entirely.

  • AsyncLocalStorage Routing — Node.js transparently switches MySQL connection pools per request lifecycle without modifying a single line of business logic.
  • Sub-Second Provisioning — TenantProvisioningService creates all 21 tables and seeds Super Admin credentials in under 800ms on checkout.
  • LRU Connection Pool Cleanup — TenantConnectionManager automatically evicts idle pools after 30 minutes, optimizing server memory in production.
  • JWT Tenant Identity Binding — Access tokens embed tenantSlug + tenantDbName so every subsequent API call is cryptographically scoped to the correct isolated database.

FAQ

Questions? We've Got Answers.

How fast is database provisioning after checkout?
Our automated TenantProvisioningService creates your dedicated database, executes all 21 system tables, and initializes your Super Admin account in under one second right when checkout completes. No waiting, no setup emails.
Is my company's data physically separate from other companies?
Yes, completely. We do not co-mingle your records into a shared table. Every workspace receives its own MySQL database instance (e.g., tenant_your_company). A SQL query from Workspace A cannot ever access data from Workspace B — it's physically impossible by design.
Can I upgrade or downgrade my subscription plan later?
Absolutely. Upgrading instantly modifies your platform subscription tier without any database migration, immediately unlocking advanced modules like attendance tracking, 2FA enforcement, and the custom RBAC builder.
What happens to my data if I cancel?
We retain your complete database backup for 90 days after cancellation. You can export a full MySQL dump at any time through the Super Admin panel. Your data is always yours.
Does NexusTask support Single Sign-On (SSO)?
TOTP 2FA (compatible with Microsoft Authenticator and Google Authenticator) is available on all plans. SAML/Okta SSO integration is available on Enterprise plans through our custom API integration option.
